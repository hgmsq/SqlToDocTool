# SqlToDocTool
这是一款简单的数据库文档生成工具，主要实现了 SQlServer+MYsql 说明文档的生成，并且支持 SQLServer 数据库的备份功能，主要可以把数据库的表以及表的详细字段信息导出到 Word 中，可以方便开发人员了解数据库的信息或写技术说明文档。
